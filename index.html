Frequently Asked Questions

What to do if the app crashes on startup?

Please ensure your system version meets the minimum requirements (macOS 14.0+) and try restarting the application. If the problem persists, please contact technical support.

How to import SVG files?

In the application, select "File" > "Import SVG", then choose your SVG file to import it into the canvas.

What file formats are supported?

Currently supports SVG format import and export, as well as application-specific project file formats.
